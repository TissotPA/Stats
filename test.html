Test de l'application Basket Stats

Instructions

1. Ouvrez la console du navigateur (F12)

2. Cliquez sur le lien ci-dessous

3. Allez dans l'onglet "Évolution"

4. Sélectionnez une joueuse

5. Cochez une ou plusieurs statistiques

6. Vérifiez les messages dans la console

Lien vers l'application

→ Ouvrir l'application Basketball Stats

Points à vérifier

  • ✅ L'onglet "Évolution" est présent dans la navigation
  • ✅ Le sélecteur de joueuse se remplit correctement
  • ✅ Les checkboxes de statistiques apparaissent après sélection d'une joueuse
  • ✅ Le graphique s'affiche quand on coche une statistique
  • ✅ Le tableau d'historique se remplit avec les matchs de la joueuse

Messages de debug dans la console

Si le système ne fonctionne pas, vérifiez ces messages dans la console :

  • Evolution player changed: [ID]
  • Updating matches table for player: [ID]
  • Available matches: [nombre]
  • Match [adversaire]: [données joueur]
